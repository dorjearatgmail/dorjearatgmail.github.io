Build Your Website Prototype Workshop

5-Hour Intensive Training for No-Code Development

1 Introduction to No-Code Development

Learning Outcomes

  • Understand what no-code development is
  • Overview of builder.io and bolt.new
  • Explore use cases and examples

Activities

  • Discussion: Why no-code?
  • Platform demos: builder.io and bolt.new

2 Building with builder.io

Learning Outcomes

  • Navigate builder.io interface
  • Create pages and components
  • Add interactivity and animations
  • Manage content effectively

Activities

  • Hands-on: Build a landing page with builder.io
  • Exercise: Implement a simple contact form

3 Building with bolt.new

Learning Outcomes

  • Compare bolt.new with builder.io
  • Utilize drag-and-drop editor
  • Customize templates for unique designs
  • Ensure mobile responsiveness

Activities

  • Hands-on: Recreate a design using bolt.new
  • Exercise: Optimize for mobile devices

4 Deployment to Netlify

Learning Outcomes

  • Connect project to Git repository
  • Set up continuous deployment with Netlify
  • Configure custom domains
  • Understand performance optimization basics

Activities

  • Hands-on: Deploy a prototype to Netlify
  • Exercise: Add a custom domain

5 Final Project & Q&A

Learning Outcomes

  • Integrate learned skills into a complete prototype
  • Identify and troubleshoot common issues
  • Understand best practices for no-code development

Activities

  • Guided final project build
  • Open Q&A session
