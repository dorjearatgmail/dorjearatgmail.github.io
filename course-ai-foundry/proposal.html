Azure AI Foundry Workshop

3-Hour Intensive Training for AI Engineers

1 Introduction & Foundations

Learning Outcomes

  • Understand what Azure AI Foundry is and what AI agents are
  • Review agent capabilities, architecture, and real-world use-cases

Activities

  • Quick poll: What problems do agents solve in your workplace?
  • Short video/demo: Building an AI Agent Using Azure AI Foundry

2 Hands-On Setup & First Agent Build

Learning Outcomes

  • Set up your Azure AI Foundry environment
  • Build your first agent using the Azure portal

Activities

  • Guided hands-on exercise: Log in to Azure portal
  • Create project and build basic question-answering agent

3 Customizing & Integrating Agents

Learning Outcomes

  • Upload files and documents for agent knowledge base
  • Connect to Azure AI Search for enhanced document retrieval
  • Add custom actions and workflows to extend agent capabilities

Activities

  • Code lab: Build a code-first agent
  • Enhance agent with plugins/tools

4 Advanced Scenarios & Q&A

Learning Outcomes

  • Understand enterprise features and monitoring
  • Explore advanced uses like RAG and image generation

Activities

  • Demo of monitoring/tracing dashboard
  • Live Q&A session

5 Wrap-up & Takeaways

Resources

  • Microsoft Learn modules
  • GitHub labs and workshop guides

Completion

  • Feedback survey
  • Certificates of completion
